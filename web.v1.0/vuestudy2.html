vue条件语句

2019.11.25

1.v-if and v-else

随机生成一个数字，判断是否大于0.5，然后输出对应信息。

//链接文件，之后每个文件均包含此行
<script src="https://cdn.staticfile.org/vue/2.2.2/vue.min.js"></script>

<div id="app">
    <div v-if = "Math.random()>0.5">
        Sorry
    </div>
    <div v-else>
        Not sorry
    </div>
</div>
<script>
    new Vue({
        el:'#app'
    })
</script>

2.v-else-if

v-else-if在2.1.0中新增，顾名思义，用作v-if的else-if块,可以链式的多次使用。

<div id="app2">
     <div v-if = "type === 'A'"></div>
     A 
     <div v-else-if = "tyle === 'B'"></div>
     B 
     <div v-else-if = "type === 'C'"></div>
     C 
     <div v-else></div>
     Not A B C
    </div>
<script>
    new Vue({
        el:'app2',
        data:{
            type:'C'
        }
    })
</script>
