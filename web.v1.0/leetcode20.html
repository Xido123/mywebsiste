1.分析

今天开通了leetcode的账户，并尝试了两道题，第一道为#1，太基础就不写笔记了， 第二道为#20 题目链接 第二道做了差不多两小时，根本原因是数据结构的基础太差，也是好几个月没看了，今天开始刷leetcode 提高自己的数据结构水平，现在就先不谈算法水平了。 额，现在来分析一下题目，大意为输入一个字符串如"()","(]""(())",括号类型为 '('，')'，'{'，'}'，'['，']'
要求： 1.左括号必须用相同类型的右括号闭合。2.左括号必必须以正确的顺序闭合。
思路：
由于左括号与右括号在我的认知里比较难比较，所以使用-1对应（ ,1对应） ，这样就可以通过和为零容易的得出结果。 1，将括号依次放进栈，当有一个新括号进来的时候存在栈顶，并与栈中上一个加入的元素进行比较; 这样最后如果栈不为空，就说明括号不能一一抵消，则返回false，否则返回true。 

package ccc;
//构造一个栈来装括号所对应的数据;数据类型为int
public class Stack {
    private int[] array;
    private int maxSize;
    private int top;
    public Stack(int size) {
        this.maxSize = size;
        array = new int[size];
        top = -1;
    }
    //推入栈顶
    public void push(int value) {
        if(top<maxSize-1) {
            array[++top] = value;
        }
    }
    //伪弹出栈顶， 其实是把top作为栈顶游标下移一个位置。
    public void pop() {
            top--;
    }
    
    public static boolean isValue(String s) {
        Stack stack = new Stack(s.length()+1);
        boolean b = false;
        
            for(int i = 0 ;i <= s.length()-1;i++) {
                //因为括号类型有限，所以用对应的数字代替简化
                char c = s.charAt(i);
                if(c == '(')
                stack.push(-1);
                else if(c == ')')
                    stack.push(1);
                else if(c == '[')
                    stack.push(-2);
                else if(c == ']')
                    stack.push(2);
                else if(c == '{')
                    stack.push(-3);
                else {
                    stack.push(3);
                }
                if(stack.top>=1 && 
                (stack.array[stack.top]+stack.array[stack.top-1] == 0)) {
                    stack.pop();
                    stack.pop(); 
                    
                }
            }
        if(stack.top == -1) {
            b = true;
        }
        return b;
    }
