vue循环语句

2019.11.26

1.v-for指令

循环使用v-for指令
v-for指令需要用到 site in sites 形式的特殊语法sites: 元数据组 site：数组元素迭代 

<div id="app">
<ol>
    <li v-for = "site in sites">
        {{site.name}}
    </li>
</ol>
</div>
<script>
new Vue({
    el: 'app',
    data: {
        sites: [
            { name: 'Vue'},
            { name: 'React'},
            { name: 'Ang'}
        ]
    }
})
</script>
2.v-for在模板中使用
<div id="app2">
<ul>
<template v-for="site in sites">
    <li> {{site.name}} </li>
    <li>-------------------</li>
</template>
</ul>
</div>
<script>
new Vue({
    el:'#app2',
    data: {
    sites: [
      { name: 'Runoob' },
      { name: 'Google' },
      { name: 'Taobao' }
    ]
  }
})
</script>
2.v-for迭代对象

V-FOR可以通过一个对象的属性来迭代对象数据

<div id = #app3>
    <ul>
        <li v-for= "value in object"></li>
        {{value}}
    </ul>
<script>
    new Vue({
        el:'#app3',
        data:{
            object:{
                name:'百度',
                url:'https://baidu.com',
                slogan:'百度一下你就知道'
            }
        }
    })
</script>
 </div>
3.v-for迭代器提供多个参数为键名

<div id="app4">
<ul>
    <li v-for="(key,value,index) in object">
        {{value}}.{{key}}.{{index}}
    </li>
</ul>
<script>
new Vue({
    el:'#app4',
    data: {
        object:{
            name:'百度',
            url:'baidu.com',
            slog:'百度你好'
        }
    }
})
</script>
</div>
3.v-for迭代整数

<div id="app5">
<ul>
    <li v-for="n in 10">
       {{n}}
    </li>
</ul>
<script>
new Vue({
    el:'#app5'  
})
</script>
</div>
